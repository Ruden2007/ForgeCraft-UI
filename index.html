Класс SelectList представляет собой список с возможностью выбора элемента. Основные возможности класса включают::

  1. Автоматически отображает активный элемент списка и устанавливает его как выбранный.
  2. Поддерживает опцию 'isRequired', которая определяет, должен ли быть выбран хотя бы один элемент (по умолчанию true).
  3. Генерирует событие 'change' при изменении выбранного элемента.
  4. Поддерживает добавление и удаление элементов.
  5. Обновляет список при добавлении или удалении элементов.
  6. Позволяет устанавливать выбранный элемент.

Класс WindowTabs предоставляет удобный интерфейс с вкладками и окнами для управления контентом. Возможности класса включают:

  1. Создание вкладок с соответствующими окнами контента.
  2. Установка активной вкладки и отображение соответствующего окна.
  3. Добавление и удаление вкладок и их связанных окон.
  4. Позволяет управлять видимостью вкладок и окон.
  5. Предоставляет событие 'changeWindow' при изменении активного окна.
  6. Поддержка управления с клавиатуры (при включенной опции).
Dropdown

Класс Dropdown обладает различными функциональностями:

  1. Позволяет создавать выпадающий список на основе элемента select.
  2. Предоставляет возможность отображать текст-плейсхолдер, когда ни один элемент не выбран.
  3. Позволяет устанавливать максимальное количество элементов в выпадающем списке без необходимости прокрутки.
  4. Позволяет устанавливать и получать значения выбранного элемента.
  5. Позволяет устанавливать и получать значения выбранного элемента.
  6. Автоматически обновляет стили выпадающего списка в зависимости от настроек, включая динамическую проверку прокрутки и позиционирование списка относительно кнопки.
  7. Дает возможность изменять выбранный вариант с использованием колесика мыши.
  8. Позволяет запрещать выбор определенных вариантов в списке.
