Класс SelectableItems предоставляет следующие возможности для управления выбираемыми элементами в контейнере:

  1. Управление выбором элементов:
    • Поддержка выбора одного или нескольких элементов внутри контейнера.
    • Возможность настройки обязательного выбора хотя бы одного элемента с помощью опции isRequired.
    • Возможность настройки множественного выбора с помощью опции multiple.
  2. Навигация и управление клавиатурой:
    • Поддержка навигации с клавиатуры с использованием настраиваемых клавиш (по умолчанию 'ArrowUp', 'ArrowDown', 'Enter').
    • Обработка клавиатурных событий для переключения активных элементов и подтверждения выбора.
  3. Создание визуального представления списка:
    • Добавление класса 'selectable' к контейнеру списка.
    • Добавление класса 'selectable-item' к элементам списка.
  4. Методы:
    • Метод toggleItem(item) позволяет переключать состояние выбора для элемента.
    • Метод getSelectedItems() возвращает массив выбранных элементов.
    • Метод setSelectedItem(indexOrElement) позволяет устанавливать выбранный элемент по его индексу или HTMLElement.
    • Обработка события 'listUpdate', которое возникает при обновлении списка элементов.
  5. Привязка событий:
    • Вызов события 'change' при выборе элемента.
    • Вызов события 'listUpdate', которое возникает при обновлении списка элементов.

Класс SelectList предоставляет следующие возможности:

  1. Создание списка с возможностью выбора элементов:
    • Инициализация нового объекта SelectList для управления выбором элементов внутри элемента select.
    • Возможность настройки обязательного выбора хотя бы одного элемента с помощью опции isRequired.
    • Возможность настройки множественного выбора с помощью атрибута multiple у select.
  2. Навигация и управление клавиатурой:
    • Поддержка навигации по элементам с клавиатуры с использованием настраиваемых клавиш (по умолчанию 'ArrowUp', 'ArrowDown', 'Enter').
    • Методы для программного изменения выбранного элемента и обработки клавиатурных событий.
  3. Свойства класса:
    • Получение и установка текущего выбранного элемента выпадающего списка с помощью свойства value.
    • Получение выбранных элементов списка с помощью свойства selectedItems.
  4. Создание визуального представления списка:
    • Скрытие оригинального элемента select и создание пользовательского визуального элемента выпадающего списка.
    • Добавление класса 'select-list' к созданному контейнеру выпадающего списка.
  5. Привязка событий:
    • Вызов события 'change' при выборе элемента.
    • Установка выбранного элемента в select при изменении активного элемента в пользовательском выпадающем списке.
Dropdown

Класс Dropdown предоставляет следующие возможности:

  1. Создание выпадающего списка на основе элемента select:
    • Инициализация нового объекта Dropdown для управления выбором элементов внутри элемента select.
    • Отображение текста-плейсхолдера, когда ни один элемент не выбран.
    • Установка максимального количества элементов в выпадающем списке без необходимости прокрутки.
    • Автоматическое обновление стилей выпадающего списка в зависимости от настроек, включая динамическую проверку прокрутки и позиционирование списка относительно кнопки.
  2. Навигация и управление:
    • Возможность изменять выбранный вариант с использованием колесика мыши, если курсор над не открытым списком.
  3. Методы:
    • Установка и получение значений выбранного элемента.
    • Запрещение выбора определенных вариантов в списке.
  4. Привязка событий:
    • Вызов события 'change' при выборе элемента.
    • Установка выбранного элемента в select при изменении активного элемента в пользовательском выпадающем списке.

Класс StackedPages предоставляет следующие возможности:

  1. Управление вкладками и окнами:
    • Добавление новых страниц в стек вкладок с помощью метода addPage(content, name).
    • Установка контента для страницы по индексу или имени с помощью метода setPageContent(indexOrName, content).
  2. Навигация с клавиатуры:
    • Переключение активных страниц с помощью клавиш "ArrowRight" и "ArrowLeft".
    • Клавиши можно настроить с помощью nextKey и prevKey
  3. Выбор активных элементов:
    • Возможность выбора одного или нескольких активных элементов.
    • Обязательный выбор активного элемента, если опция isRequired установлена в true.
  4. Управление классами:
    • Автоматическое добавление класса 'page' ко всем дочерним элементам контейнера вкладок.
    • Автоматическое добавление класса 'tab-window' ко всем страницам.
  5. Управление страницами через атрибут data-name:
    • Переключение активных элементов по значению атрибута data-name с помощью метода toggleActiveElement(indexOrName).
  6. Расширенные настройки:
    • Возможность настройки навигации с клавиатуры (enableKeyboardNavigation).
    • Возможность выбора нескольких активных элементов (multiple).
